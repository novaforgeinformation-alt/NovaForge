# NovaForge

Landing page for NovaForge — a software engineering agency. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript (strict)
- **Styling:** Tailwind CSS
- **Fonts:** Inter, Manrope, JetBrains Mono via `next/font`
- **Icons:** Material Symbols Outlined
- **Email:** Nodemailer + Gmail SMTP

## Structure

```
app/          — Routes and API (contact, terms, privacy)
components/   — Section components (Nav, Hero, Services, etc.)
lib/          — Shared utilities (fonts, contact context)
public/       — Static assets (images)
```
